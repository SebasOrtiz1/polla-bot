# 🏆 Bot Polla Mundial 2026 — WhatsApp

## Variables de entorno en Railway
ANTHROPIC_API_KEY=sk-ant-...   ← Tu key de Anthropic
TWILIO_AUTH_TOKEN=...           ← De tu cuenta Twilio (opcional, para validación)

## Comandos del bot
- **hoy** → Partidos del día con pronósticos
- **tabla** → Puntos actuales U1 vs U2
- **resultado A1 2-1** → Ingresar resultado real
- **pronostico A1** → Ver pronóstico de un partido
- **pendientes** → Partidos sin resultado
- **ayuda** → Ver todos los comandos
- Cualquier pregunta libre → Responde Claude con contexto del torneo

## IDs de partidos
- Grupos: A1-A6, B1-B6, C1-C6, D1-D6, E1-E6, F1-F6,
          G1-G6, H1-H6, I1-I6, J1-J6, K1-K6, L1-L6
- Octavos: O1-O16
- Cuartos: Q1-Q7
- Semis: S1-S4
- Final: FN1
